\# Foundation Project – Enterprise VM Buildout



\## Overview



This project documents the creation of the foundational virtual machine environment used throughout the Enterprise Systems Support Portfolio.



The objective was to build a repeatable Windows lab environment capable of supporting Active Directory administration, enterprise support exercises, troubleshooting scenarios, and help desk simulations.



This environment serves as the foundation for all subsequent portfolio projects.

\## Lessons Learned



1\. Proper lab preparation reduces troubleshooting later.

2\. Snapshots provide rapid recovery during experimentation.

3\. Consistent documentation simplifies portfolio development.

4\. Building the environment is the first step toward mastering enterprise administration.
