# Foundation Project – Enterprise VM Buildout

## Project Summary

This project established the virtualized lab environment used throughout my Enterprise Systems Support Portfolio.

The purpose of this buildout was to create a stable foundation for practicing Windows administration, Active Directory, networking, troubleshooting, documentation, and enterprise support workflows.

This environment serves as the base system for future labs involving Active Directory, Group Policy, help desk support, networking, and osTicket service desk projects.

---

## Environment Overview

The lab environment was built using a virtualized setup that supports multiple enterprise-style systems.

- Screenshot-based validation

## Lessons Learned

This project reinforced the importance of preparing a stable technical foundation before building more advanced labs.

The main lessons learned were:

- Proper lab organization makes future projects easier to manage.
- Snapshots are important for testing and recovery.
- Documentation should be created while building, not after the fact.
- A consistent folder structure makes GitHub easier for others to follow.
- A strong VM foundation supports repeatable enterprise support practice.
