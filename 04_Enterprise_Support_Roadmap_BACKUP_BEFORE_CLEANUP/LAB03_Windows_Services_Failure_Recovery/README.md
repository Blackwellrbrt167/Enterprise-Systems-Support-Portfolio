# LAB 03 · Windows Services Failure Recovery

## Overview

This lab simulates a real-world enterprise service outage involving a critical Windows service that has stopped unexpectedly or repeatedly crashes.

The objective is to diagnose the failure, analyze dependencies, review recovery settings, inspect event logs, remediate the issue, validate restoration of service, and document the incident according to enterprise support standards.

---

## Skills Demonstrated

* Windows Services Administration
* Service Recovery Configuration
* Service Dependency Analysis
* Event Viewer Investigation
* PowerShell Troubleshooting
* Root Cause Analysis
* Ticket Documentation
* Enterprise Escalation

---

## Technologies Used

* services.msc
* PowerShell
* sc.exe
* Event Viewer
* Service Control Manager

---

## Business Scenario

A shared print server experiences recurring Print Spooler failures during periods of heavy usage.

Users are unable to print and Help Desk tickets continue to increase.

---

## Lab Objectives

* Identify failed service
* Review service configuration
* Analyze dependencies
* Review recovery settings
* Restart and validate service
* Review Event Viewer evidence
* Configure automatic recovery
* Validate restoration
