# Static IP Addressing and Subnetting



## Overview



This lab focuses on configuring static IPv4 addressing, validating network connectivity, understanding subnetting fundamentals, and troubleshooting common network configuration issues.



## Objectives



- Configure static IP addresses

- Understand subnet masks

- Identify network and broadcast addresses

- Validate DNS and gateway settings

- Troubleshoot connectivity issues

- Document root cause analysis



## Environment



### Domain Controller



- Windows Server 2022

- Active Directory Domain Services (AD DS)

- DNS Server

- DHCP Server

- IP Address: 192.168.10.10



### Client Workstation



- Windows Client

- Domain Joined

- Static IP Configuration



## Validation Checklist



- \[ ] Static IP configured

- \[ ] Subnet mask validated

- \[ ] Default gateway validated

- \[ ] DNS server configured

- \[ ] Connectivity testing completed

- \[ ] Subnetting calculations completed

- \[ ] Root cause analysis documented
